PLC/SLICED (SLICE) Bel Documentation

SLICE bels contain two LUT4s, two D flipflops and carry logic (CCU2). They also contain a MUX2 driven by the LUT4s to create a LUT5.

Bel Pins

PinWire
A0← JA0_SLICEDLUT0 A input
A1← JA1_SLICEDLUT1 A input
B0← JB0_SLICEDLUT0 B input
B1← JB1_SLICEDLUT1 B input
C0← JC0_SLICEDLUT0 C input
C1← JC1_SLICEDLUT1 C input
D0← JD0_SLICEDLUT0 D input
D1← JD1_SLICEDLUT1 D input
M0← JM0_SLICEDFF0 M (direct) input
M1← JM1_SLICEDFF1 M (direct) input
DI0← JDI0_SLICEDFF0 DI (LUT/mux output) input
DI1← JDI1_SLICEDFF1 DI (LUT output) input
SEL← JSEL_SLICEDLUT MUX2 select input
FCI← JFCI_SLICEDfast carry input
CLK← JCLK_SLICEDFF clock
CE← JCE_SLICEDFF clock enable
LSR← JLSR_SLICEDFF local set/reset
F0→ JF0_SLICEDLUT0 output
F1→ JF1_SLICEDLUT1 output
Q0→ JQ0_SLICEDFF0 output
Q1→ JQ1_SLICEDFF1 output
OFX0→ JOFX0_SLICEDMUX2 output
FCO→ JFCO_SLICEDfast carry out
